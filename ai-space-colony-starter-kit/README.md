# AI Space Colony Starter Kit

Starter kit for `Mhaizza/ai-space-colony-sim`.

This kit contains:

- GitHub labels
- GitHub milestones
- Initial backlog issues
- Import scripts using GitHub CLI
- AI Studio prompts, workflows, templates, and docs structure

Recommended import order:

1. Import labels
2. Import milestones
3. Import issues
4. Add imported issues to your GitHub Project board

See `github-starter-kit/import/README.md`.
