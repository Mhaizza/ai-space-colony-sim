# Kanban Update Protocol

## Mandatory Rules

No Issue → No Work.

No Kanban Update → Not Done.

## Before Starting

The agent must identify:

- Kanban card
- Goal
- Owner
- Affected files
- Acceptance criteria

## Completion Format

```txt
Kanban Update

Card:
Status:
Completed:
Changed Files:
Validation:
Follow-up Tasks:
```
